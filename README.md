## 🚀 Project Overview: OmniRouteOmniRoute is an advanced trade-copying architecture that decouples the MetaTrader 5 terminal from the execution logic. By using a FastAPI-based bridge, it allows for complex "Many-to-Many" routing—assigning specific slave accounts to follow different master accounts simultaneously.This project serves as a robust MVP in a DevOps ecosystem, focusing on low-latency execution and high-availability monitoring. 

### ⚡ Key FeaturesMulti-Master Routing: Link multiple MT5 Master accounts to specific Slave clusters using a relational backend.  Decoupled Architecture: Master terminals push signals via WebRequests to a centralized Python bridge, reducing terminal overhead.Real-time Telegram Auditing: Get instant HTML-formatted notifications for trade execution, equity shifts, and connection heartbeats.Electric Blue Dashboard: A modern, dark-mode management interface for monitoring account health and assigning followers on the fly.  Dynamic Lot Sizing: Automated lot calculation based on Master/Slave equity ratios to ensure precise risk management. 

### 🛠️ Tech StackBackend: Python 3.10+, FastAPI, SQLAlchemy.Database: SQLite (Local) / PostgreSQL (Production).Frontend: HTML5, Tailwind CSS (Electric Blue Theme).Integration: MetaTrader 5 Python API, Telegram Bot API.
