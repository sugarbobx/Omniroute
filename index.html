⚡ Quick Setup ▾
🔵 Why "Failed to Fetch"?
The dashboard needs the Python bridge server running. Start it first:
1On your VPS: pip install -r requirements.txt
2Run: python bridge/main.py
3Enter the VPS IP below, e.g. http://192.168.1.10:8000
4Click Connect → red light turns blue ✓
🔑 API Key Setup
In your .env file set API_SECRET=yourkey, then paste the same key in the field below. All requests will send it as X-API-Key header.
Bridge Config ▾
Bridge URL
?
The address of your running Python bridge. Use http://127.0.0.1:8000 if running locally, or http://YOUR_VPS_IP:8000 for a remote server.
API Key
?
Optional security key. Set API_SECRET in your .env file on the server. Paste the same value here to authenticate all dashboard requests.
Add MT5 Account ▾
Terminal Path
?
Full path to terminal64.exe on your Windows VPS. Default works for standard MT5 installs.
Magic Number *
?
Set this same number in your Master EA's Magic Number input. When the EA fires a signal with this number, the bridge routes it to linked slaves.
Global Symbol Map ▸
Translates symbol names between brokers. Example: Master uses GOLD, Slave broker calls it XAUUSD — add a mapping so orders reach the correct instrument.
